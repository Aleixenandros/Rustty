Rustty

última release · Linux · macOS · Windows

Terminal y gestor de conexiones para SSH, SFTP y RDP.

Ligero, privado y open source. Construido con Rust y Tauri, Rustty reúne terminal, panel SFTP, túneles SSH, sesiones divididas, KeePass/keyring y temas configurables en una interfaz rápida.

Instalación rápida

Instala Rustty

Usa el instalador automático o descarga binarios precompilados para Linux, Windows y macOS.

# rustty install

# Instalador automático: detecta tu distro $ curl -sSf https://rustty.es/install.sh | sh
# O manualmente: Fedora / openSUSE / RHEL $ sudo dnf install ./Rustty-{version}-1.x86_64.rpm
# Debian / Ubuntu / Mint $ sudo apt install ./Rustty_{version}_amd64.deb
# Arch / Manjaro $ sudo pacman -U Rustty-{version}-1-x86_64.pkg.tar.zst
# AppImage portable $ chmod +x Rustty_{version}_amd64.AppImage && ./Rustty_{version}_amd64.AppImage
Características

Todo lo que necesitas.

Seguridad local, flujos técnicos rápidos y control sobre tus conexiones.

--ssh --sftp --rdp

Multi-protocolo

SSH, SFTP y RDP en la misma aplicación. Backend 100% Rust vía russh y russh-sftp, sin dependencia de libssh2.

--sftp-panel

Panel SFTP integrado

Explorador de ficheros con drag & drop, que sigue el cwd del terminal y permite elevar a sudo cuando el servidor lo permite.

--tunnels -L -R -D

Túneles SSH

Redirecciones locales, remotas y SOCKS sobre una sesión activa, con panel de estado, tráfico y autoconexión por perfil.

--split --broadcast

Multi-pestaña y splits

Split horizontal, vertical o rejilla. Broadcast de teclas para teclear en varias sesiones simultáneas.

--keepass --keyring

KeePass y keyring

Contraseñas desde una base .kdbx o desde el keyring del sistema: KWallet, GNOME Keyring, Keychain, Credential Store.

--backup --sync

Copias cifradas

Sincroniza perfiles, preferencias, temas y atajos con Google Drive, iCloud Drive, carpeta local/NAS o WebDAV. Todo viaja cifrado con age.

--themes --fonts

Temas y tipografía

11 temas base, tema del terminal independiente del de UI, selector de fuente monoespaciada y export/import de temas en JSON.

--keybindings

Atajos configurables

Editor de atajos en Preferencias con captura en vivo, detección de conflictos y restauración por acción.

--i18n

Multiidioma

Interfaz en español, inglés, francés y portugués. Detecta automáticamente tu locale.

--open-source

Open source

Licencia Apache-2.0. Código, issues y releases públicos en GitHub.
