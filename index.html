Opciones avanzadas

0 = desactivado. Envía un paquete keepalive al servidor cada N segundos para evitar caídas por NAT.

⚠️ Añade aes-cbc, dh-sha1, hmac-sha1 y ssh-rsa a la negociación. Reduce la seguridad: úsalo solo con servidores antiguos.

Reusa el socket del agente local ($SSH_AUTH_SOCK) para autenticar saltos desde el host remoto.

Requiere un X server local en localhost:6000 (DISPLAY=:0). En Windows o macOS necesitarás un servidor X dedicado.

0 = desactivado. Número de reintentos automáticos al caer la conexión, con backoff exponencial (2s, 4s, 8s…).

Conecta primero a este host bastión y abre un canal SSH al destino real a través suyo (ProxyJump). Reutiliza las credenciales del perfil.

Vuelca la salida del shell a un fichero de log dentro de session_logs/<perfil>-<timestamp>.log en el directorio de datos.

Sincronización en la nube

Sincroniza perfiles, preferencias, temas y atajos entre equipos. Cifrado de extremo a extremo (E2E) con tu passphrase. El servidor solo ve datos opacos.

Puedes apuntar aquí a una carpeta local, NAS o carpeta sincronizada por el sistema.

~/Library/Mobile Documents/com~apple~CloudDocs/Rustty/

Usa iCloud Drive como carpeta local sincronizada. Disponible en macOS con iCloud Drive activo.

Se guarda en el keyring del SO con servicio rustty, clave sync:webdav_password.

No conectado

Cifra los datos antes de subirlos. Sin esta passphrase nadie (ni siquiera el administrador del servidor) puede leer tus perfiles. Si la pierdes, pierdes acceso a la sincronización. Se guarda en el keyring del SO.

Nunca se sincroniza el keyring del SO ni la base KeePass desbloqueada.
